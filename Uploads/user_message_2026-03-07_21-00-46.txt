O certificado deve ser renderizado exatamente igual ao HTML original — tamanho fixo A4 landscape (1122×794px), com ondas verdes, bordas douradas, cantos ornamentais, medalha por nível, Excelino, assinatura e QR Code.
Passo 1 — Copiar a imagem do Excelino para o projeto:
Salvar o arquivo certificado-arena-excel.html que foi gerado localmente. Abrir o arquivo, localizar a tag:
html<img class="excelino-img" src="data:image/png;base64,....
Copiar o valor inteiro do src (incluindo o prefixo data:image/png;base64,) e salvar em public/excelino-certificado.png usando o seguinte script Node.js:
js// scripts/extract-excelino.js
const fs = require('fs');
const html = fs.readFileSync('./certificado-arena-excel.html', 'utf8');
const match = html.match(/class="excelino-img" src="(data:image\/png;base64,[^"]+)"/);
if (match) {
  const base64 = match[1].replace('data:image/png;base64,', '');
  fs.writeFileSync('./public/excelino-certificado.png', Buffer.from(base64, 'base64'));
  console.log('✅ Excelino extraído para public/excelino-certificado.png');
}
Executar com: node scripts/extract-excelino.js

export const getServerSideProps: GetServerSideProps = async ({ params }) => {
  try {
    const res = await fetch(
      `https://arenaexcel.excelcomjohni.com.br/api/badges/certificado/validar/${params?.id}`
    );
    if (!res.ok) return { props: { valido: false, id:'', nomeAluno:'', curso:'', nivel:'', dataConquista:'' } };

    const cert = await res.json();
    const dataFormatada = new Date(cert.dataConquista).toLocaleDateString('pt-BR', {
      day: '2-digit', month: 'long', year: 'numeric'
    });

    return {
      props: {
        valido: true,
        id: cert.id,
        nomeAluno: cert.nomeAluno,
        curso: cert.curso,
        nivel: cert.nivel,
        dataConquista: dataFormatada,
      }
    };
  } catch {
    return { props: { valido: false, id:'', nomeAluno:'', curso:'', nivel:'', dataConquista:'' } };
  }
};
